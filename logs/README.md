## ðŸ§­ Overview

This project implements a **production-grade ETL pipeline** that automatically extracts stock market data from the **Yahoo Finance API**, processes it using **Apache Spark**, and loads it into a **MySQL data warehouse**, orchestrated daily by **Apache Airflow**.

It demonstrates the design principles of **modular data engineering**, **scalable computation**, and **cloud-ready orchestration**, with full testing, logging, and visualization support.
